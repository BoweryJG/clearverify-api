ClearVerify Widget Integration

This example demonstrates how to integrate the ClearVerify insurance verification widget into your dental practice website.

Live Demo

Click the "Check Insurance Coverage" button in the bottom-right corner to try the widget.

Example Procedures:

  • D6010 - Surgical placement of implant body
  • D6065 - Implant supported porcelain/ceramic crown
  • D0120 - Periodic oral evaluation

Integration Code

Add this code to your website to enable insurance verification:

<!-- Add before closing </body> tag --> <script src="https://api.clearverify.com/widget/v1/embed.js"></script> <script> ClearVerify.init({ apiKey: 'your-api-key-here', providerId: 'your-provider-id', procedureCodes: [ { code: 'D6010', description: 'Surgical placement of implant body' }, { code: 'D6065', description: 'Implant supported crown' }, { code: 'D0120', description: 'Periodic oral evaluation' } ], onSuccess: function(result) { console.log('Verification successful:', result); // Handle successful verification }, onError: function(error) { console.error('Verification failed:', error); // Handle errors } }); </script>

Customization Options

ClearVerify.init({ // Required apiKey: 'your-api-key', providerId: 'your-provider-id', // Optional theme: 'light', // or 'dark' position: 'bottom-right', // or 'bottom-left', 'top-right', 'top-left' // Procedure codes specific to your practice procedureCodes: [ { code: 'D6010', description: 'Implant placement' }, // Add more procedures ], // Callbacks onSuccess: function(result) { // result contains coverage details }, onError: function(error) { // Handle errors } });
